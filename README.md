# FORJA — Science-Based Hypertrophy Engine

A personal fitness PWA for muscle hypertrophy with progressive overload tracking, posture correction, and science-based programming.

## Features
- 10 unique full-body workouts (2-week rotation, Mon-Fri)
- Science-based progressive overload (double progression method)
- Smart suggestions: tells you when to increase weight, build reps, or deload
- Muscle group color coding (no consecutive same-muscle exercises)
- Posture correction warm-ups for kyphosis (with HOW-TO guides)
- Exercise technique guides with detailed instruction images
- Progressive cardio programming (6 phases from base building to sport-ready)
- Built-in rest timer
- Week/month navigation to see past and future routines
- Inline workout logging
- All data stored locally on your device (private)

## Based On
- Jeff Nippard's evidence-based hypertrophy principles
- Schoenfeld 2016 meta-analysis
- Helms et al. 2014 progressive overload for beginners

## How to Use
1. Open the app URL in Safari on your iPhone
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Start training!
